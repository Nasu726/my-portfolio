## About this site
ここではマイページの概要についてまとめています。

### Home
ホームページです。\
ページ上部には、簡単な自己紹介と興味関心のキーワードが書かれています。\
「Readme」ボタンを押すとこのページに遷移します。\
「My Career」ボタンを押すとプロフィールページに遷移します。\
「Contact」ボタンを押すとメールが起動します(確認ダイアログが出ます)。\
プロジェクトとブログを新しい方から3つずつピックアップして、カード形式で表示しています。上から新しい順に並んでいます。\
カード上部にはバッジが付いていて、リリースから30日以内のプロジェクト/ブログには「New」バッジが付いています。\
カード下部のタグをクリックすると、同じタグが付いたプロジェクト/ブログを絞り込んで検索できます。

### Readme
このページです。マイページの概要についてまとめています。

### Projects
今までに作成したプロジェクトをカードにしてまとめています。上から新しい順に並んでいます。\
カードをクリックするとプロジェクト(またはGitHubリポジトリ)にジャンプします。\
カード上部にはバッジが付いていて、リリースから30日以内のプロジェクトには「New」バッジが付いています。\
カード下部のタグをクリックすると、同じタグが付いたプロジェクトを絞り込んで検索できます。

### Blog
今までに書いたブログをカードにしてまとめています。上から新しい順に並んでいます。\
カードをクリックするとブログ本文を読めます。\
カード上部にはバッジが付いていて、リリースから30日以内のブログには「New」バッジが付いています。\
カード下部のタグをクリックすると、同じタグが付いたブログを絞り込んで検索できます。

### My Career
プロフィール、経歴、スキルについてまとめています。

### Contact
公式の連絡先です。クリックでメールが起動します(確認ダイアログが出ます)。
