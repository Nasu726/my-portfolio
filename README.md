## About this site
ここではマイページの概要についてまとめています。

### Home
ホームページです。\
ページ上部には、簡単な自己紹介と興味関心のキーワードが書かれています。\
「Readme」ボタンを押すとこのページに遷移します。\
「My Career」ボタンを押すとプロフィールページに遷移します。\
「Contact」ボタンを押すとメールが起動します(確認ダイアログが出ます)。\
プロジェクトとブログを新しい方から3つずつピックアップして、カード形式で表示しています。上から新しい順に並んでいます。\
カード上部にはバッジが付いていて、リリースから30日以内のプロジェクト/ブログには「New」バッジが付いています。\
カード下部のタグをクリックすると、同じタグが付いたプロジェクト/ブログを絞り込んで検索できます。

### Readme
このページです。マイページの概要についてまとめています。

### Projects
今までに作成したプロジェクトをカードにしてまとめています。上から新しい順に並んでいます。\
カードをクリックするとプロジェクト(またはGitHubリポジトリ)にジャンプします。\
カード上部にはバッジが付いていて、リリースから30日以内のプロジェクトには「New」バッジが付いています。\
カード下部のタグをクリックすると、同じタグが付いたプロジェクトを絞り込んで検索できます。

**※注意**
<ul>
  <li>利用時には各プロジェクトごとの注意書きやライセンス表記に従ってください。</li>
  <li>ブログやSNSでの紹介のための引用は自由にしていただいて構いません。ただし、作者名やリンクなどが分かるように明記してください。</li>
</ul>


### Blog
今までに書いたブログをカードにしてまとめています。上から新しい順に並んでいます。\
カードをクリックするとブログ本文を読めます。\
カード上部にはバッジが付いていて、リリースから30日以内のブログには「New」バッジが付いています。\
カード下部のタグをクリックすると、同じタグが付いたブログを絞り込んで検索できます。

**※注意**
<ul>
  <li>ブログやSNSでの紹介のためおよび参考資料としての引用は自由にしていただいて構いません。ただし、引用箇所や参考文献欄などにブログリンクを明記してください。</li>
</ul>

### Books
購入した本(積読含む)をカードにしてまとめています。\
カードをクリックすると読書メモを読めます。\
書影下部または読書メモ上部のボタンから、出版社やAmazonのページにジャンプできます。\
カード上部にはステータスバッジが付いていて、読書の進行状況やおすすめ度が見られます。\
カード下部のタグをクリックすると、同じタグが付いた本を絞り込んで検索できます。\

**※注意(著作権に関して)**
<ul>
  <li>書影は以下のものを使用しています。</li>
  <ol>
    <li>版元ドットコム<a href="https://www.hanmoto.com" class="underline" target="_blank">(https://www.hanmoto.com)</a>で【利用可】と書かれているもの。</li>
    <li>1.にあてはまらず、自分で撮影したもの。</li>
  </ol>
  <li>読書メモが主体である本文にあたり、その本の紹介および一意性(出版社などの区別)の確保を目的として書影を載せています。</li>
  <li>積読ページという性質上、未読や読みかけの本については本文が非常に短く、本文と引用部の主従関係が不明確な場合があります。「積読」ステータスなどの明記をしておりますので、意図的な主従関係の転倒でないことにご注意ください。</li>
  <li>本棚形式のディスプレイはサイト内の検索性を向上させるためのものであり、むやみに書影を掲載してページを装飾するといった意図はありません。</li>
</ul>
