## Brainfuckビジュアライザを公開

C++の勉強を兼ねて、Brainfuckのビジュアライザを作成・公開しました。

- ビジュアライザのページは **[こちら](https://algoviz.nasu.uk)**
- GitHubリポジトリは **[こちら](https://github.com/Nasu726/AlgoViz)**

Brainfuckは難解プログラミング言語(esolang)の一つとして知られています。命令が非常に単純であるがゆえに書きづらく、それを乗り越えても高級言語同様のプログラムを書くことは非常に困難です。そこで、Brainfuckの**実行時の動作を可視化**し、**変数宣言の機能を追加**してアドレスを追いやすくすることで快適なBrainfuckコーディングが体験できるようなビジュアライザを作成しました。

## AlgoVizの紹介

AlgoVizは様々なアルゴリズムやデータ構造、機械の動作を可視化するビジュアライザを統合したWebアプリケーションとなることを目指しています。VisuAlgoの完全日本語版+αを一つの目標として、これからもビジュアライザを追加する予定です。

### 構成

- React (Vite)
- C++ (Emscripten)

### Brainfuckビジュアライザの主な機能

#### コントロール
- コード編集
- 標準入力 / 標準出力
- 実行 / 一時停止
- ステップ実行 / ステップバック
- 実行速度変更（インターバル: 0〜1秒）
- ポインタの自動追従 / カメラの手動制御

#### テープ
- セルの値の表示
- セルの値をUTF-8として見たときの文字の表示
- セルのアドレスの表示
- ポインタ位置と実行ステップ数の表示

#### その他
- ヘルプの設置（仕様説明・文法・キーボードショートカット）
- コードの保存
